= smf

smf is a library for SMF (Standard MIDI File).

== Usage

[source,nim]
----
import smf

var s = newSMF()

var midiEvent1 = newMIDIEvent(0, statusOn)
s.add midiEvent1

writeFile("a.smf", s)
----
